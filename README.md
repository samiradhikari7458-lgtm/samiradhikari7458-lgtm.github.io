# Samir Adhikari Portfolio

A professional, responsive, and modern developer portfolio website showcasing skills, projects, and contact information.

## Technologies Used
* HTML5
* CSS3 (Grid & Flexbox)
* Git & GitHub Pages

## Features
* **Responsive Layout:** Optimized for mobile and desktop.
* **Modern UI:** Features a layered hero section with an overlapping profile.
* **UX Focused:** Includes scroll-to-top functionality and smooth entrance animations.

## Live Demo
You can view the live site at: https://samiradhikari7458-lgtm.github.io/

## How to use
This site is built with static files. Simply edit the `index.html` or `css/style.css` files and push to the `main` branch to update the site.
